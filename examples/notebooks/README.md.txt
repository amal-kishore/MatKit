# Examples for `matkit`: ABC-Type Dataset

This example suite demonstrates the full pipeline for materials with ABC-type stoichiometry using `matkit`.

## Structure

- `scripts/`: Step-by-step .py scripts (can run individually or via `run_all.sh`)
- `notebooks/`: One-stop Jupyter notebook (`ABC_workflow_demo.ipynb`)
- Output files (CSVs, plots) are generated in the local folder

## How to Use

### Option 1: Run Scripts
```bash
cd examples/scripts
chmod +x run_all.sh
./run_all.sh
